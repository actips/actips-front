# actips-front

这是 ACTips.org for ACMer's 平台的前端项目代码，采用 TypeScript + Vuejs 开发。

这是纯前端 SPA(Single Page App) 项目，与后端交互纯粹通过 RESTful 接口。

对于 ACTips 产品的建议，统一在本项目的 Issues 中交流，作者会在看到之后尽快回复，感谢你们的参与。

感谢 Star，欢迎 Fork。
